# LetGo-LetSave
## A mobile app concept designed to help users discover, track, and share local deals securely. The app focuses on providing a centralized platform for users to save, organize, and receive notifications for local personalized offers.
## Issue 
Consumers often miss out on limited-time deals because they are scattered across multiple platforms and stores. There is no easy way to track, save, or verify these offers efficiently.
## Solution
Deal Finder solves this problem by allowing users to:
- Create personal accounts and secure profiles
- Save favorite deals for quick access
- Receive real-time notifications for new deals
- Post and share verified deals with the community
  ## Tech Stack
  Frontend: React Native
- Backend: Node.js
- Database: MySQL
- Authentication: JWT
  ## Features
- User Registration & Secure Login
- Deal Posting & Verification
- Favorites Tracking
- Real-Time Notifications
## Future Improvements
- AI-based deal recommendations based on user preferences
- Fraud detection and deal verification improvements
- Social sharing features to increase community engagement
